- **[中華民國偽娘交流協會](https://www.facebook.com/ROCTransgenderSociety/)**

- **[Trif 的勸世酒吧](https://www.instagram.com/triftransbar/)**

- **[踏踏福神](https://portaly.cc/Tatamascot)**  
  時常舉辦「女裝日」,提供女裝相關嘗試、交流的空間與機會
